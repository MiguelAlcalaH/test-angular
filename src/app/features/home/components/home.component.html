Verisure Angular Quickstart

Welcome to the Verisure Angular Quickstart v{{ version }} with MSAL v2 Authentication!!

The objective of this quickstart is to provide a scalable base template based on Angular framework.

This sample demonstrates how to configure MSAL to login, logout, protect routes ("/api-health" and "/profile"), and acquire an access token for a protected resources such as the Microsoft Graph API and API Backend.

Please review the project's README.md file for details regarding the libraries and tools used in this Angular quickstart. You can also find guides related to best practices and troubleshooting in Angular development.

This quickstart is experimental. If you have any comments or suggestions, please contact the Solution Architecture Team. Thank you.
